üîß Kenapa Harus Pakai Operator?

Di soal gaji & lembur, kamu diminta:

‚Ä¢ Menghitung total gaji = gaji pokok + gaji lembur

üßÆ Operator Aritmatika (Perhitungan Gaji)

+ (penjumlahan) untuk menghitung total gaji:

Operator lain yang sering dipakai:

  • - (pengurangan)
  • / (pembagian)
  • % (modulo / sisa bagi)

‚öñÔ∏è Operator Perbandingan (Digunakan di IF)

Dipakai untuk membandingkan nilai, misal jam lembur atau golongan.

  • == (sama dengan)
  • >= (lebih besar sama dengan)
  • <= (lebih kecil sama dengan)
  • < (lebih kecil)
  • > (lebih besar)
  • != (tidak sama dengan)

Contoh dipakai di soal lembur:

Tanpa operator, program tidak bisa:
‚ùå Menghitung gaji pokok, gaji lembur, dan total gaji
‚ùå Membandingkan nilai (jam lembur, golongan, dll)
‚ùå Membuat logika perhitungan yang sesuai dengan ketentuan soal

üîÄ Kenapa Harus Pakai IF?

Di soal, banyak kalimat berbunyi "Jika...", misalnya:

  • Jika Golongan A: Gaji Rp. 5.000.000
  • Jika Golongan B: Gaji Rp. 6.500.000
  • Jika Golongan C: Gaji Rp. 9.500.000
  • Jika lembur 1 jam ‚Üí 30%
  • Jika lembur 2 jam ‚Üí 32%
  • Jika lembur 3 jam ‚Üí 34%
  • Jika lembur 4 jam ‚Üí 36%
  • Jika lembur ‚â• 5 jam ‚Üí 38%
Kalimat "Jika ‚Ä¶ maka ‚Ä¶" = logika percabangan.
Di Java, percabangan dibuat dengan: if, else if, else.

  • ‚úÖ Memilih gaji sesuai golongan (A, B, C)

üìä Kenapa Harus Pakai Array?

Di soal tertulis jelas:
‚Ä¢ Gaji setiap golongan disimpan pada Array Gaji
‚Ä¢ Persen lembur disimpan pada Array lembur

Artinya, dosen memang ingin kamu belajar menyimpan beberapa nilai yang sejenis dalam satu variabel (array), bukan membuat banyak variabel terpisah.

‚ùå Tanpa Array (lebih ribet)

Setiap kali mau ambil gaji/lembur, kamu harus bikin IF lagi. Lebih panjang dan rawan salah.

‚úÖ Dengan Array (lebih rapi & hemat kode)

int[] gaji = { 5_000_000, 6_500_000, 9_500_000 }; // index 0 = A, 1 = B, 2 = C double[] persenLembur = { 0.30, 0.32, 0.34, 0.36, 0.38 }; // index 0 = 1 jam, 1 = 2 jam, dst.

Cukup sekali deklarasi, lalu tinggal akses pakai index.

üéØ Cara Akses Data dari Array

Untuk Gaji Golongan:

  • Golongan A ‚Üí index 0 ‚Üí gaji[0]
  • Golongan B ‚Üí index 1 ‚Üí gaji[1]
  • Golongan C ‚Üí index 2 ‚Üí gaji[2]

Untuk Persen Lembur:

  • 1 jam ‚Üí persenLembur[0] (30%)
  • 2 jam ‚Üí persenLembur[1] (32%)
  • 3 jam ‚Üí persenLembur[2] (34%)
  • 4 jam ‚Üí persenLembur[3] (36%)
  • ‚â• 5 jam ‚Üí persenLembur[4] (38%)
Array dipakai karena:
‚úÖ Data yang disimpan sejenis (semua gaji / semua persen lembur)
‚úÖ Jumlahnya lebih dari satu dan punya pola index (A=0, B=1, C=2; jam=1‚Äì5)
‚úÖ Mempermudah akses: cukup pakai index, bukan banyak variabel terpisah

üìù Ringkasan

üîß Operator

Dipakai untuk menghitung gaji pokok, gaji lembur, dan total gaji, serta membandingkan nilai (jam lembur, golongan, dll).

üîÄ IF Statement

Dipakai untuk mengambil keputusan:
‚Ä¢ Golongan mana yang dipakai
‚Ä¢ Persentase lembur mana yang digunakan
‚Ä¢ Input valid atau tidak

üìä Array

Dipakai untuk menyimpan banyak data sejenis:
‚Ä¢ Gaji per golongan (A, B, C)
‚Ä¢ Persen lembur (30, 32, 34, 36, 38)

üéØ Kesimpulan Akhir

Ketiga konsep ini wajib dipakai di soal gaji & lembur karena:

‚úÖ Soal secara eksplisit meminta penggunaan Array
‚úÖ Ada logika "jika ... maka ..." yang memerlukan IF
‚úÖ Ada perhitungan gaji dan lembur yang memerlukan Operator

Tanpa Operator, IF, dan Array, program tidak bisa menyelesaikan kasus gaji lembur dengan benar.
