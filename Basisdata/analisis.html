0. Persiapan Data

Wajib dilakukan sebelum analisis a–e

# Paket yang diperlukan
library(readxl)
library(dplyr)
library(ggplot2)
library(ggpubr)

# Baca file
data_raw <- read_excel(file.choose())

# Rapikan data
data <- data_raw %>%
  slice(-1) %>%
  transmute(
    Pendapatan  = as.numeric(...2),
    Kebahagiaan = as.numeric(...3)
  )

# Cek hasil
str(data)
head(data)

b. Scatter Plot Kebahagiaan vs Pendapatan

Code (b)

# Scatter plot Kebahagiaan (X) vs Pendapatan (Y)

Scatter plot pada Gambar 2 menampilkan hubungan antara Kebahagiaan dan Pendapatan. Titik-titik data cenderung membentuk pola yang menaik dari kiri bawah ke kanan atas, artinya ketika nilai Kebahagiaan meningkat, Pendapatan juga cenderung ikut meningkat. Pola ini mengindikasikan adanya hubungan linear positif antara Kebahagiaan dan Pendapatan, sehingga penggunaan model regresi linear sederhana cukup sesuai untuk menggambarkan hubungan kedua variabel tersebut.

c. Regresi Linear Sederhana

Code (c)

# Model regresi linear sederhana
model <- lm(Pendapatan ~ Kebahagiaan, data = data)

# Ringkasan hasil model
summary(model)

Intercept

0.91927

Koefisien Kebahagiaan

1.04608

R-squared

0.7452

Adjusted R-squared

0.7447
Ŷ = 0.91927 + 1.04608 × Kebahagiaan

Penjelasan (c)

Berdasarkan hasil regresi linear sederhana diperoleh persamaan di atas. Koefisien Kebahagiaan sebesar 1,04608 berarti bahwa setiap kenaikan 1 satuan skor Kebahagiaan akan meningkatkan Pendapatan rata-rata sekitar 1,046 satuan, dengan asumsi faktor lain tetap. Nilai p-value yang sangat kecil (p < 0,001) menunjukkan bahwa Kebahagiaan berpengaruh signifikan terhadap Pendapatan. Nilai R-squared sebesar 0,7452 mengindikasikan bahwa sekitar 74,52% variasi Pendapatan mampu dijelaskan oleh Kebahagiaan dalam model, sedangkan sisanya dijelaskan oleh faktor lain di luar model.

d. Pemeriksaan Homoskedastisitas

Code (d)

# Plot diagnostik khusus Residuals vs Fitted
plot(model, which = 1)

Penjelasan (d)

Grafik Residuals vs Fitted digunakan untuk mengecek asumsi homoskedastisitas, yaitu apakah varians residual konstan di seluruh rentang nilai prediksi. Pada grafik, titik-titik residual tampak menyebar di sekitar garis nol, baik di atas maupun di bawah, tanpa pola “kipas” yang jelas. Meskipun garis halus sedikit menurun pada nilai fitted yang tinggi, sebaran titik secara umum masih acak. Oleh karena itu, dapat disimpulkan bahwa asumsi homoskedastisitas secara umum cukup terpenuhi pada model regresi linear sederhana Pendapatan terhadap Kebahagiaan, sehingga model masih layak digunakan untuk keperluan analisis dan interpretasi.

e. Visualisasi Hasil Regresi

Code (e)

# Visualisasi: titik data + garis regresi + persamaan garis
ggplot(data, aes(x = Kebahagiaan, y = Pendapatan)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  stat_regline_equation(
    aes(label = ..eq.label..),
    label.x = 1.5, label.y = 7.5
  ) +
  labs(
    title = "Hubungan antara Kebahagiaan dan Pendapatan",
    x     = "Kebahagiaan",
    y     = "Pendapatan"
  ) +
  theme_minimal()

Penjelasan (e)

Gambar 4 menampilkan scatter plot antara Kebahagiaan dan Pendapatan yang dilengkapi dengan garis regresi linear. Titik-titik pada grafik merepresentasikan data observasi, sedangkan garis regresi menggambarkan pola hubungan rata-rata yang diperkirakan oleh model. Kemiringan garis yang positif konsisten dengan koefisien Kebahagiaan yang juga bernilai positif, sehingga menguatkan kesimpulan bahwa peningkatan Kebahagiaan cenderung diikuti oleh peningkatan Pendapatan. Persamaan garis regresi ditampilkan langsung di dalam grafik sehingga pembaca dapat melihat bentuk hubungan matematis tanpa harus membuka output numerik di konsol. Judul dan label sumbu X–Y memperjelas informasi yang disajikan dan membuat grafik lebih mudah dipahami.
